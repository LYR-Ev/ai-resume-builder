# AI Resume Builder

现代化 Web 简历制作器，基于 Next.js 14 + TypeScript + Tailwind CSS 构建。

## 功能

- JSON / Markdown / TXT 导入
- 左侧编辑 + 右侧实时预览
- 3 套可切换模板（简约 / 现代蓝 / 深色高级）
- 导出 JSON / Markdown / PDF
- localStorage 自动保存
- 一键填充示例简历与重置
- mock AI 文案优化（可扩展至真实 API）
- 深色/浅色模式切换

## 本地运行

```bash
npm install
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000)。
